Spendful

Spendful is a calm, minimal daily spend awareness mobile application. It's deliberately NOT a budgeting app - instead, it focuses on mindful awareness through a single daily question: "Did you spend money today?" The app follows an ethical, non-manipulative design philosophy with a calm, non-judgmental tone throughout. It's designed to be offline-first with local SQLite storage, no authentication required, and minimal external dependencies.
